# Portofolio Profesional — Fayiz

Website portofolio statis dengan desain elegan (dark + gold accent), animasi hover, dan section lengkap.

## Menjalankan

Buka `index.html` di browser, atau jalankan server lokal:

```bash
cd portfolio
npx serve .
```

## Sesuaikan dengan data Anda

1. **Nama & bio** — edit teks di `index.html` (hero, about, CV).
2. **Foto** — `../img/fayiz.jpeg`
3. **Medsos** — ubah `href` pada link GitHub, Instagram, LinkedIn, dll.
4. **Email** — ganti `hello@fayiz.dev`.
5. **Proyek** — tambah/ubah kartu di section `#projects`.
6. **CV PDF** — letakkan `cv.pdf` di folder ini dan set `href="cv.pdf"` pada tombol unduh.

## Struktur

- `index.html` — markup & konten
- `styles.css` — desain & hover effects
- `script.js` — navigasi aktif, filter proyek, animasi scroll
